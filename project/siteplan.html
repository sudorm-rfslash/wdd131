Site Plan for My Web Project

Site Name

Name: Cinechque

Reason for Selection: I wanted to create a website to 'check' information about 'cinema'. 'Cine' is also spanish for cinema.

Optional Domain: cinecheque.com, chinecheque.org

Optional Variations: cine-cheque.com, chine-cheque.org

Site Purpose

My website will let users browse top movies and view basic information about them. Good for seeing what's new in theatres.

Scenarios

Examples of questions a site visitor might have:

Where do you pull your website information from?:
  • I plan to use The Movie DataBase (TMDB) to get my movie info.
Why should I use this website isntead of XYZ movie site?
  • I want my site to be designed clean and clutter free. There shouldn't be any distractions like ads or other unnecessary information.

Color Schema

Primary Color: #007acc (used in headings and accents)

Secondary Color: #555555 (used in body text)

Typography

Font Choice: I will use Roboto to match all my other websites I've done this semester.
